# DaniBot Legal & Support

Welcome to the official legal and support page for **DaniBot**.

DaniBot is a Discord bot that allows users to connect their own Spotify and YouTube Music accounts to use account-based music features inside Discord servers.

## Legal Documents
- [Privacy Policy](./privacy.html)
- [Terms of Service](./terms.html)
- [Support](./support.html)

## What DaniBot does
- OAuth account connection (Spotify / YouTube Music)
- Music-related features in Discord
- Account disconnection on user request

## Data & Security Summary
- OAuth tokens are used only to provide requested features.
- Users can disconnect at any time using bot commands.
- DaniBot does not sell user data.
