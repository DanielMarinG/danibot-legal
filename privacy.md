# Privacy Policy - DaniBot

DaniBot allows users to connect Spotify/YouTube Music accounts for Discord features.

## Data we process
- OAuth access tokens and refresh tokens
- Basic profile info (display name/email when available)

## How we store data
- Tokens are stored in memory for bot operation.
- We do not sell personal data.

## User rights
You can disconnect anytime using bot commands (`/spotify disconnect`, `/ytmusic disconnect`).
